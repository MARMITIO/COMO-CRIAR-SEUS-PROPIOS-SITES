Como Criar um Site em HTML

HTML (Linguagem de Marcação de Hipertexto) é a base de qualquer site. Para criar um site simples, você só precisa de um editor de texto (como o Bloco de Notas ou o VS Code) e de um navegador.

Exemplo Básico de HTML:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Meu Primeiro Site</title>
</head>
<body>
    <h1>Bem-vindo ao Meu Primeiro Site</h1>
    <p>Este é um exemplo simples de um site em HTML.</p>
</body>
</html>

Este código cria uma página com um título e um parágrafo. Basta salvar o arquivo com a extensão .html (por exemplo, index.html) e abri-lo em um navegador para visualizar o site.

Como Publicar seu Site na Web

Agora que você já criou seu site, pode publicá-lo na web para que outras pessoas possam acessá-lo. Aqui estão os passos básicos:

1. Escolha um Serviço de Hospedagem

Para que seu site esteja disponível online, você precisará de um serviço de hospedagem. Algumas opções populares são:

  • GitHub Pages - Grátis, ideal para sites simples.
  • Netlify - Grátis e fácil de configurar.
  • HostGator - Opção paga com mais recursos.

2. Envie seu Arquivo HTML

Na maioria dos serviços, basta enviar seus arquivos HTML para o servidor de hospedagem. No caso do GitHub Pages, por exemplo, você pode fazer isso criando um repositório e enviando os arquivos do seu site.

3. Conecte um Domínio (Opcional)

Se você quiser um endereço personalizado para seu site (como www.meusite.com), você pode registrar um domínio em serviços como GoDaddy ou Namecheap e conectá-lo à sua hospedagem.
